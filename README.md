# Expense Analysis Dashboard

## Overview
This project focuses on data cleaning, expense analysis, and dashboard reporting using business expense data.

## Features
- Data Cleaning
- Expense Trend Analysis
- Dashboard Visualization
- KPI Reporting
- Business Insights

## Tools Used
- Excel
- Power BI
- GitHub
- Canva

## Objective
To analyze company expense data and generate insights for better business decision-making.
